Create images from [.tBin file](https://doc.mapeditor.org/en/stable/manual/export-tbin/) and stylesheets

## Usage
1. Put .tbin file in folder "in/"
2. Put stylesheets in folder "stylesheets/"
3. Run script
4. See result in folder "out/"
